# Linux-Server Einrichtung

## Umgebung
- Ubuntu Server in einer virtuellen Maschine (UTM) installiert  
- System für IT-Administrationsübungen genutzt  

## Durchgeführte Schritte
- Ubuntu Server installiert  
- Virtuelle Maschine konfiguriert  
- Im Linux-System angemeldet  
- SSH-Installation überprüft  

## Verwendete Grundbefehle
- whoami → aktuellen Benutzer anzeigen  
- hostname → Systemnamen anzeigen  
- ssh -V → SSH-Version prüfen  

## Ergebnis
Eine funktionierende Linux-Server-Umgebung für praktische Übungen erfolgreich eingerichtet
