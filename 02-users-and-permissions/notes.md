# Benutzer und Berechtigungen – Notizen

## Wichtige Konzepte

### Benutzer
Jede Person, die ein Linux-System nutzt, ist ein Benutzer mit einer eindeutigen ID.

### Gruppen
Benutzer können Gruppen zugeordnet werden, um Berechtigungen effizient zu verwalten.

### Berechtigungen
Linux verwendet ein 3-stufiges Berechtigungssystem:
- Besitzer (Owner)
- Gruppe (Group)
- Andere (Others)

Berechtigungen:
- r = Lesen
- w = Schreiben
- x = Ausführen

### Wichtige Befehle
- adduser → Benutzer erstellen  
- id → Benutzerinformationen anzeigen  
- groups → Gruppenmitgliedschaften anzeigen  
- chmod → Berechtigungen ändern  
- chown → Besitzer ändern
