# Home IT Infrastructure Lab

## Überblick
Dieses Projekt simuliert eine grundlegende IT-Infrastruktur mit einem Linux-Server in einer virtualisierten Umgebung. Es zeigt grundlegende Fähigkeiten in der Systemadministration.

## Umgebung
- Ubuntu Server (virtuelle Maschine in UTM)
- macOS Host-System

## Was ich gelernt habe

- Linux-Systemadministration  
- Benutzer- und Gruppenverwaltung  
- Dateiberechtigungen (chmod, chown)  
- Grundlagen von Netzwerken (IP, Verbindung)  
- System-Monitoring-Tools  
- Simulation von Dateifreigabe und Zugriffskontrolle  
- Grundlagen von SSH  

## Projektstruktur

- 01-linux-server → Einrichtung und Systemkonfiguration  
- 02-users-and-permissions → Benutzerverwaltung und Zugriffskontrolle  
- 03-networking → Netzwerkgrundlagen  
- 04-file-sharing-simulation → Simulation einer Unternehmensstruktur  
- 05-system-monitoring → Systemüberwachung und Performance-Tools
